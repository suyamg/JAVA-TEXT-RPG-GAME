# (1) 요구 사항

Java Development Kit (JDK): JDK 8 이상

Gradle: Gradle 6.8 이상

Git: 최신 버전

운영 체제: Windows, macOS, Linux

# (2) 설치 및 실행
```
git clone https://github.com/suyamg/JAVA-TEXT-RPG-GAME.git
cd JAVA-TEXT-RPG-GAME
```
```
gradle build
gradle run
```
